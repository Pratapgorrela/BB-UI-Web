# Beauty Bus 🚐

Beauty Bus is a modern beauty service booking platform that brings salon-quality beauty treatments directly to your doorstep. Users can easily browse services, book appointments, manage schedules, and enjoy a premium salon experience from the comfort of their homes.

The platform connects customers with a fully equipped mobile salon service, allowing them to book haircuts, spa treatments, beauty consultations, and more at their preferred time and location.

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js (v18+ recommended)
- npm or yarn

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/beauty-bus.git
```

Navigate to the project:

```bash
cd beauty-bus
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Application will be available at:

```text
http://localhost:5173
```
